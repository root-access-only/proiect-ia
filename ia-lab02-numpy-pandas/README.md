# Proiect IA - Laborator 02: NumPy si Pandas

Laborator de Inteligenta Artificiala. Exercitii practice cu NumPy, Pandas, Matplotlib si Seaborn.

## Structura

- `cerinta_3_1_numpy.py` — operatii de baza cu NumPy: creare array-uri, statistici, operatii vectorizate, comparatie performanta liste vs NumPy
- `cerinta_3_2_pandas.py` — explorare si manipulare date cu Pandas pe dataset-ul Iris: filtrare, agregare, coloane calculate
- `cerinta_3_3_matplotlib.py` — grafice Matplotlib (linie, scatter, bare, histograma) pe dataset-ul Iris
- `cerinta_3_4_seaborn.py` — vizualizarea distributiilor cu Seaborn: histograma cu KDE, boxplot, violinplot
- `cerinta_3_5_heatmap.py` — heatmap corelatie Pearson pe dataset-ul Iris
- `cerinta_3_6_titanic copy.py` — analiza supravietuirii pe dataset-ul Titanic: rate per sex, clasa, vizualizari
- `tema/tema_a_numpy.py` — tema NumPy: operatii matriceale, produs, inversa, determinant
- `tema/tema_b_tips.py` — tema Pandas: analiza dataset-ului Tips, bacsisuri, grupari
- `tema/tema_c_dashboard.py` — tema Dashboard: figura 2x2 cu scatter, boxplot, histograma, barplot pe Tips
- `tema/tema_d_bonus_iris.py` — bonus: pairplot si violinplot-uri pe dataset-ul Iris
- `tema/tema_e_colab.ipynb` — bonus: notebook Jupyter cu analiza Titanic
